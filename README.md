## PIEKARNIK UV
Aktualna wersja piekarnika pozwala na korzystanie z pięciu programów grzania. Urządzenie wyposażone jest z wyłącznik krańcowy, aby zapobiec włączeniu naświetlania z otwartymi drzwiami.
Obudowa urządzenia składa się z podwójnych ścian, które pełnią funkcję kanałów chłodzących dla radiatorów, na których przylutowane są diody uv.
